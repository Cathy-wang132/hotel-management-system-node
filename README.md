# 酒店管理系统-node+express+vue

#### 1、介绍

这是一个完整的酒店管理系统项目，后端：Node.js 16.20 以上 + Express 框架，数据库使用 MySQL 8.0，前端：使用 Vue 2 + Element UI，通过本项目，可以掌握完整的前后端开发流程，学习常见的开发框架和工具，提升技术水平！

#### 3、系统功能

· 首页

· 酒店管理

· 楼宇管理

· 房间管理

· 会员管理

· 开房登记

· 登记管理

· 设备维修

· 安全检查

· 管理员管理

#### 4、技术栈与工具

● 后端：Node.js 16.20 以上 + Express 框架，数据库使用 MySQL 8.0，构建工具采用 npm

● 前端：使用 Vue 2 + Element UI

● 开发工具：使用 VSCode

● 环境要求：前端 Node.js 版本 16.20 以上，Node.js 16.20 以上，MySQL 8.0，npm
